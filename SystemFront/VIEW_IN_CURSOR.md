# Cursor 안에서 앱 보기

개발 서버(`npm run dev` 또는 `npm run next:dev`)가 **실행 중**일 때, Cursor IDE 안에서 바로 페이지를 보는 방법입니다.

## 방법: Simple Browser 사용

1. **명령 팔레트** 열기: `Ctrl+Shift+P` (Mac: `Cmd+Shift+P`)
2. **"Simple Browser: Show"** 입력 후 선택
3. URL 입력란에 아래 주소 입력 후 Enter  
   **`http://localhost:3000`**

이렇게 하면 Cursor 창 안에 Simple Browser 패널이 열리고, 해당 주소가 표시됩니다. 외부 브라우저를 쓰지 않아도 됩니다.

---

**참고**: 서버가 꺼져 있으면 먼저 터미널에서 `npm run dev`(또는 `npm run next:dev`)를 실행하세요.
